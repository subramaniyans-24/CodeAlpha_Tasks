CodeAlpha Machine Learning Internship Projects

This repository contains the Machine Learning projects completed as part of the CodeAlpha Internship Program. Each project focuses on solving real-world problems using Machine Learning and Deep Learning techniques.

A Machine Learning model that predicts the creditworthiness of loan applicants based on financial and personal attributes.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Joblib

Features

- Feature Encoding
- Model Training
- Credit Risk Prediction
- Performance Evaluation
