# Credit Scoring Model

## Project Overview

This project was developed as part of the CodeAlpha Machine Learning Internship.

The objective of this project is to predict an individual's creditworthiness using historical financial data and machine learning techniques.

---

## Dataset

Credit Risk Dataset

Features include:

- Person Age
- Person Income
- Home Ownership
- Employment Length
- Loan Intent
- Loan Grade
- Loan Amount
- Interest Rate
- Credit History Length

Target Variable:

- loan_status

---
